# ☕ Brûlé — Specialty Coffee Website

A modern, animated coffee shop website built using **HTML, CSS, and JavaScript**.
The design focuses on a luxury café aesthetic with smooth animations, interactive menu filtering, and a stylish UI.

## ✨ Features

* 🎨 Modern coffee shop design
* 🖱 Custom animated cursor
* 📜 Smooth scroll navigation
* 🍰 Interactive menu filtering (Espresso, Filter, Cold, Food)
* 🛒 Simple cart indicator system
* 📱 Responsive layout for mobile and desktop
* ✉ Newsletter subscription interaction
* 🎬 Scroll reveal animations

## 🛠 Technologies Used

* **HTML5**
* **CSS3** (Advanced styling & animations)
* **JavaScript** (Vanilla JS)
* **Google Fonts**
